## ✨ 核心特性

- **流程节点**: 发起节点、审批节点、条件分支节点、并行节点、结束节点
- **审批操作**: 提交、同意、拒绝、撤销、转交、加签、退回
- **字段权限控制**: 按节点配置字段的读写权限
- **审批人配置**: 指定成员、直属上级、部门主管、角色成员
- **审批页面定制**: 支持自定义审批页面
- **业务集成**: 支持在审批流程中调用业务逻辑元素和 AI 元素
